heroku-buildpack-exiftool
=========================

A Heroku buildpack that installs Phil Harvey's [ExifTool](http://www.sno.phy.queensu.ca/~phil/exiftool/).

Usage
-----

Add this Git repo to your Heroku buildpacks list.
